# Vision &bull; coderadi

> An app for project visualization and online collaboration.

This app is one of my oldest dreams, my purpose to build this app is to help everyone who has some projects and needs/wants to collaborate with other people, basically this type of scenario happens mostly with entrepreneurs and professionals working one a project.

## Confirmed Integrations
###### (for the pre-release version)

### 1. Project management
This integration allows users to create a project, add related tasks, and start visualizing the completion. They'll receive details like how much tasks are left, what percentage of the project is done, how much average tasks the user completed in a day, most productive day, least productive day, etc.

### 2. Public view
This integration will allow users to structure their projects so that other users can view the project. For example, if a user is creating a project (let's say lead generation prototype). So he can structure the project documentation, screenshots, and other files, just like we've GitHub README.

## Planned Integrations
###### (for future updates)

### 1. Team creation
With this integration, users can create team, and share the projects across the team members. This will increase the ease of project manage for multiple users at once, members can provide their name, bio, and pic and the app will create a "Meet the Team" section with team members details.

### 2. Community
This integration will users to search for the users with specific skills, this help users to get team members for their projects. Users can also search for projects requiring their skill-set so that they can also contribute and build their portfolio.
